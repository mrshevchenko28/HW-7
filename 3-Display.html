Спостереження номер 1. Елемент з display block займає всю ширину контейнера, елемент з display inline займає тільки ширину свого вмісту, елемент з display inline-block поводиться як inline, але дозволяє задавати ширину і висоту, елемент з display none не відображається взагалі, елемент з display flex організовує свої дочірні елементи в гнучкий контейнер, а елемент з display grid організовує свої дочірні елементи в сітку.
